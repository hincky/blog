# kafka 

Apache Kafka 是一款开源的分布式消息（Record）引擎系统。负责在不同系统之间传递语义准确的消息，实现松耦合的异步式数据传递。

1. 设计消息的格式：二进制的字节序列
2. 传输消息的方法：点对点，发布/订阅

点对点模式（消息队列）：一对一模式

发布/订阅：多个发布者向消息容器发送信息，多个订阅者从消息容器里获取信息
