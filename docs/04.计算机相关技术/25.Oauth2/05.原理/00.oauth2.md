# oAuth2

授权协议：就是保证第三方（软件）只有在获得授权之后，才可以进一步访问授权者的数据。

在 OAuth 2.0 的体系里面有 4 种角色：
- 资源拥有者 
- 第三方 
- 授权服务 
- 受保护资源
