## 文件和目录列表

```sh
# -F区分文件和目录，以/结尾就是目录，以*结尾就是可执行文件；
ls -F

# -a列出所有文件，以.开头就是隐藏文件
ls -a 

# -l展示更多信息
ls -l

# 过滤器，?代表单个，*代表零个或多个



```

## 日志
- tail
- head
- cat
- tac
- echo

```sh
# 从第3000行开始，显示1000行。即显示3000~3999行
cat test.log | tail -n +3000 | head -n 1000



# >意思是创建，>>是追加
```
