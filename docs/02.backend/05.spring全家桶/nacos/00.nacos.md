# Nacos

[Official Website](https://nacos.io/zh-cn/docs/what-is-nacos.html)

一个更易于构建云原生应用的动态服务发现、配置管理和服务管理平台。 Nacos 是构建以`“服务”`为中心的现代应用架构 (例如微服务范式、云原生范式) 的服务基础设施。

> Nacos 致力于帮助您发现、配置和管理微服务
- 动态服务发现
- 配置管理
- 服务管理
